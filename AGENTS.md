# Typography Update Requirements

## 1. Badge Text
Use the following Google Font for the **BADGE** text:

```css
@import url('https://fonts.googleapis.com/css2?family=Playwrite+ID:wght@100..400&display=swap');
```

**Requirements:**
- Apply the **Playwrite ID** font family.
- Preserve the existing badge styling unless otherwise specified.
- Ensure the font renders correctly across all modern browsers.

---

## 2. Name Text
Use the following Google Font for the **NAME** text:

```css
@import url('https://fonts.googleapis.com/css2?family=Merriweather:ital,opsz,wght@0,18..144,300..900;1,18..144,300..900&display=swap');
```

**Requirements:**
- Apply the **Merriweather** font family.
- Display the name in **two lines**.
- Reduce the font size compared to the current version.
- Make the font size fully responsive across all screen sizes (Desktop, Laptop, Tablet, and Mobile).
- Maintain proper line spacing and readability.

---

## 3. Description Text
Use the following Google Font for the **DESCRIPTION** section (including the custom cursor word-cycle text):

```css
@import url('https://fonts.googleapis.com/css2?family=Domine:wght@400..700&display=swap');
```

**Requirements:**
- Apply the **Domine** font family.
- Use this font for the entire description text, including the **custom-cursor word-cycle** animation.
- Keep the existing animation and functionality unchanged.
- Ensure responsive typography and consistent readability on all devices.

---

## General Notes
- Load all three fonts from Google Fonts.
- Ensure the fonts are optimized and loaded efficiently.
- Verify that the typography remains responsive and visually consistent across all breakpoints.
- Do not modify any existing functionality or animations—only update the typography as specified.
